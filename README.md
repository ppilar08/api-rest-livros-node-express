📚 API REST de Livros
API REST desenvolvida com Node.js e Express para gerenciamento de um acervo de livros em memória.

🚀 Tecnologias utilizadas
Node.js
Express
JavaScript (ES6)
📌 Funcionalidades
A API permite:

📖 Listar todos os livros
🔍 Buscar livro por ID
➕ Criar novo livro
✏️ Atualizar livro completo (PUT)
📝 Atualizar parcialmente (PATCH)
❌ Remover livro (DELETE)
🎯 Filtrar livros por gênero (Query Params)
